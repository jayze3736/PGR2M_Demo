# Text-to-Motion Generation GIFs

이 폴더에 Text-to-Motion 생성 결과 GIF 파일들을 업로드하세요.

## 파일 이름 규칙:
- CoMo 결과: `como_example1.gif`, `como_example2.gif`, ...
- Ours 결과: `ours_example1.gif`, `ours_example2.gif`, ...

## 예시:
```
como_example1.gif
ours_example1.gif
como_example2.gif
ours_example2.gif
como_example3.gif
ours_example3.gif
...
```

더 많은 예시를 추가하려면 index.html 파일에서 해당 섹션을 복사하여 붙여넣으세요.
